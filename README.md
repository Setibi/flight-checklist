# Checklist de Vuelo

App interna de checklist de misión de vuelo con exportación a PDF desde el navegador.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub, por ejemplo `flight-checklist`.
2. Sube este contenido al repositorio:
   - `index.html`
   - `README.md`
3. En GitHub, entra en `Settings` > `Pages`.
4. En `Build and deployment`, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda los cambios.

GitHub generará una URL parecida a:

```text
https://TU-USUARIO.github.io/flight-checklist/
```

## Uso

- Rellena los datos de misión.
- Completa el checklist por fases.
- Cierra el post-vuelo al finalizar.
- Pulsa `Exportar PDF` y elige `Guardar como PDF`.
- Guarda el PDF en la carpeta de OneDrive de la misión.

Los borradores se guardan en el navegador donde uses la app.
